# idook-backend

Repositório referente ao Backend do IDOOK

## Stack do Projeto

- Banco de Dados: MongoDB
- Backend: Node
- API REST: Mongoose
- Front-end: REACT JS
- Recebe um JSON dos Usuarios
- Os dados sensíveis não são armazenados no Banco de Dados
